Scraper
Run locally from your terminal (not available on cloud dashboard)
ℹ️ Scraper must run on your local machine

The scraper accesses Google Maps and JustDial, which requires local Playwright browser automation. This can't run on cloud servers.

To scrape leads:

cd D:/Agency/lead-gen-agency
source venv/Scripts/activate
python main.py

Then follow the prompts to configure scraping parameters (city, business type, source).

✅ Leads will automatically sync to MongoDB and appear in this dashboard within a few seconds
